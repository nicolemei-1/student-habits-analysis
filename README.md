# 📊 Student Habits Analysis

## Overview  
This project analyzes how student habits—such as study time, sleep, exercise, and mental health impact academic performance. Using data analysis and machine learning techniques, the goal is to identify key behavioral patterns associated with higher exam scores.

---

## Tools & Technologies  
- Python  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  

---

## Dataset  
- **Student Habits vs Academic Performance** (Kaggle)  
- Includes features such as study hours, sleep, social media usage, mental health, and exam scores  

---

## Key Insights  
- Increased **study hours** show a strong positive relationship with exam performance  
- High **social media and streaming usage** is associated with lower scores  
- **Sleep (6.5–8 hours)** and **regular exercise** correlate with better outcomes  
- Ensemble models like **Voting Classifier** and **SVC** achieved the highest accuracy (~87%)
